# 🎥 Live Object Detection & Tracking using YOLOv8 + Streamlit

This project is a **real-time object detection and tracking web app** built using:

- Streamlit
- YOLOv8 (Ultralytics)
- OpenCV
- streamlit-webrtc

It detects objects from your webcam and shows live bounding boxes, counts, and alerts.

---

## 🚀 Features

- 🎥 Live webcam object detection
- 🧠 YOLOv8 pretrained model (`yolov8n.pt`)
- 📊 Real-time object counting
- 🚨 Custom alert object detection
- 💾 Save detected frames automatically
- ⚡ Lightweight and fast processing

---

## 🛠️ Tech Stack

- Python 3.10+
- Streamlit
- Ultralytics YOLOv8
- OpenCV (headless version)
- WebRTC for live streaming
